# spring-jsp-example

## Stack
- Spring Boot 2.2.0.RELEASE
- Servlet API 2.5
- JSP, HTML, JS, CSS
- Tomcat 9

## Install
1. To clean and build maven project, use:
    ```bash
    mvn clean install
    ```
2. To create and run the Spring boot application, run the following code:
    ```bash
    $ mvn spring-boot:run
    ```
